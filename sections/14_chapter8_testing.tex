\chapter{Testing}

\section{Functionality Testing}
Functionality validation is performed using repository test modules and script-level verification.
\begin{itemize}
\item UI transitions and workflow progression (`test_ui_transitions.py`).
\item Vote persistence and storage behavior (`test_vote_storage.py`).
\item Ledger/counter correctness (`test_counting.py`, `test_ledger.py`).
\item State-machine determinism (`test_state_machine.py`).
\item Audit helper behavior and candidate I/O modules.
\end{itemize}

\section{Performance Testing}
Performance checks in prototype context focus on responsiveness and throughput under simulation:
\begin{itemize}
\item Capture-to-cast response delay in UI flow.
\item Ledger append latency for encrypted writes.
\item Script-driven multiple vote simulation (`simulate_votes.py`).
\item Tally execution time for ballot files.
\end{itemize}

\section{Database Integrity Testing}
Integrity testing is performed through ledger verification rather than SQL ACID checks:
\begin{itemize}
\item Validate sequence continuity and previous-hash linkage.
\item Detect tampering, deletion, or record re-ordering.
\item Verify cast-registry duplicate prevention behavior.
\end{itemize}

Environment and tools:
\begin{itemize}

\item Frameworks: PyQt5, cryptography, optional OpenCV/ONNXRuntime.
\item Testing: pytest and custom verification scripts (`verify_ledger.py`, `tally_votes.py`).
\item Mode: primarily offline/simulation with optional camera integration.

\section{Test Results Summary}
Summary of observed outcomes:
\begin{itemize}
\item Core functional tests for casting, storage, and tally executed through repository test set.
\item Ledger verification scripts successfully identify consistency and tamper conditions.
\item No blocker-level defects in prototype flow for documented scenarios.
\item Remaining limitations are related to production hardening and hardware certification.
\item Recommended next step: extend adversarial testing and formal verification of critical adapters.
\end{itemize}
