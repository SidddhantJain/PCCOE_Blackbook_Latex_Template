\chapter{Literature Review}

This chapter reviews selected work from the project literature set available in the B-17 collection (\textit{LIT BTech}) covering biometric e-voting, blockchain-enabled e-voting, and verifiable polling systems.

\section{Review Method}
The review used curated papers from the project folder and supplementary references from system documentation. The filtering criteria were:
\begin{itemize}
\item Relevance to electronic voting security, privacy, and verifiability.
\item Inclusion of biometric identity checks (fingerprint/palmprint/face).
\item Use of blockchain or append-only ledgers for auditability.
\item Feasibility for prototype implementation in academic environments.
\end{itemize}

Comparison parameters included authentication robustness, anti-tampering support, privacy design, computational cost, and deployment practicality.

\section{Research Gap}
The literature indicates strong isolated approaches but limited end-to-end prototypes combining accessibility, practical security controls, encrypted local storage, and blockchain-ready auditability in one modular system.

VoteGuard Pro addresses this gap by:
\begin{itemize}
\item Combining guided voter flow with security-first storage and audit logging.
\item Using cast-registry controls for duplicate-prevention in prototype context.
\item Enabling optional ML overlays for observability without binding vote eligibility decisions to ML output.
\item Providing a migration path from encrypted local ledgers to immutable chain anchoring.
\end{itemize}
